# ACEest Fitness & Gym

## Project Overview
This project is a Flask-based web application developed for ACEest Fitness & Gym as part of a DevOps assignment. It demonstrates modern DevOps practices including version control with Git and GitHub, automated testing using Pytest, containerization with Docker, and CI/CD pipeline automation using GitHub Actions and Jenkins.

## Objective
The objective of this project is to implement an end-to-end DevOps workflow for a fitness and gym management application. The solution covers local development, testing, Docker image creation, CI/CD automation with GitHub Actions, and Jenkins-based build integration.

## Tools and Technologies Used
- Python 3
- Flask
- Pytest
- Docker
- Git and GitHub
- GitHub Actions
- Jenkins

## Project Structure
- `app.py` - Main Flask application
- `test_app.py` - Pytest test cases
- `requirements.txt` - Python dependencies
- `Dockerfile` - Docker image definition
- `.github/workflows/main.yml` - GitHub Actions CI pipeline
- `README.md` - Project documentation
